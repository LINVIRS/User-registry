# User-registry（用户注册中心）

### 解决问题：

- 多个系统登录注册鉴权
- 解决重复系统登录注册问题

### 实现思路

- 部署本系统
- 任意系统在本系统配置即可
- 通过连接mysql 与redis 做到权限认定判断
- 对各个系统有对应的用户管理

### 展现形式：

- 管理页面
